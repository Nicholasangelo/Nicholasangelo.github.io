Features

Whether you're a digital nomad, a traveller, or a road tripper, Dinofii will help you discover new destinations. Define your search area as Earth, a country, or region. Then, choose the type of place you're looking for from the Select Feature menu. When you click Search, Dinofii will randomly return a matching feature in the search area you defined, as well as the closest city to that feature. Every piece of information returned is linked to more information. The feature name, country name, and feature type are all linked to search for more information. The feature's latitude and longitude coordinates will take you to a map of that location. The closest city information will take you to a map with directions from the city to the feature. The wifi icon will display the number of hotspots found in the area if that information is available, and the globe icon next to the wifi symbol will take you to a map of all known wifi hotspots in the area.

Design

Using Html, Bootstrap, and CSS we have designed a minimal and user friendly interface.

We have created a simple search bar with prepopulated features that you can search or browse.

Results are dynamicly displayed on Bootstrap Cards and stack as you continue to search.

APIs @ Play

GEONAMES

Finds global locations (points of interest, not cities) based on a single feature code.

EZCMD

Finds postal codes and nearby cities based on latitude and longitude coordinates.

WIGLY

Finds the number of WiFi Hotspots in a given postal code, including detailed mapping.

This is what led me to create the Potty Train.

Potty Trian is a Potty Time Scheduler that can be used for kids or Dogs. Just add a child name and a value indicating the frequency of which that child should visit the potty and the app will remind you when it is time for another go.
Now you can code while they play (yeah, right) with Potty Train running in the background, and not miss a potty or dog walking session.
